# Curso de midulive: notas y resúmenes

Esta sección acompaña al curso en vídeo (YouTube, canal
[midulive](https://www.youtube.com/@midulive)) ["Curso de CSS desde cero completo y GRATIS"](https://www.youtube.com/playlist?list=PLUofhDIg_38q7l8gV4IVCz_pjUeyD99_j),
compuesto por 4 clases en directo (~6,5 horas en total).

Para cada clase existe un documento de Google Docs con el resumen didáctico completo,
estructurado por capítulos, con ejemplos y referencias cruzadas a la documentación oficial.

| # | Vídeo | Duración | Resumen (Google Docs) |
|---|---|---|---|
| 1 | [¡APRENDE CSS GRATIS! Curso de CSS desde cero para principiantes](https://www.youtube.com/watch?v=hrxjBqZWsb0) | 1:43:07 | _pendiente de enlace_ |
| 2 | [¡Guía de CSS Flexbox para aprender de forma sencilla!](https://www.youtube.com/watch?v=PSwlAuRbv_A) | 1:30:56 | _pendiente de enlace_ |
| 3 | [Aprende CSS Grid MUY FÁCIL con ejercicios](https://www.youtube.com/watch?v=iTjkiI8QQsM) | 1:36:23 | _pendiente de enlace_ |
| 4 | [Aprende Animaciones con CSS y Scroll Animations sin JavaScript](https://www.youtube.com/watch?v=RwjgfNX41TE) | 1:37:05 | _pendiente de enlace_ |

!!! note "Metodología"
    Los resúmenes de las clases 1 y 2 se han redactado a partir de la transcripción literal
    del vídeo (subtítulos automáticos) más los capítulos oficiales de la descripción,
    contrastando cada explicación técnica con MDN y las especificaciones W3C.
    Los resúmenes de las clases 3 y 4 se han redactado a partir de los capítulos/título/descripción
    del vídeo y de documentación oficial, ya que la transcripción automática de YouTube no
    estuvo disponible para esas dos clases en el momento de escribir esta documentación.
